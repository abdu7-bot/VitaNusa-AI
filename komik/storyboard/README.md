# Storyboard Komik

Folder ini digunakan untuk menyimpan susunan panel dan alur visual komik VitaNusa AI.

Isi yang cocok disimpan di sini:

- pembagian panel
- deskripsi adegan
- ekspresi karakter
- posisi kamera/angle
- catatan visual untuk desainer atau AI image generator

Gunakan storyboard untuk memastikan pesan edukasi tersampaikan dengan jelas, ramah, dan tidak menakut-nakuti.
