# Prompt Gambar Komik

Folder ini digunakan untuk menyimpan prompt AI untuk membuat gambar komik VitaNusa AI.

Isi yang cocok disimpan di sini:

- prompt karakter
- prompt tiap panel
- prompt gaya visual
- prompt banner komik
- catatan konsistensi warna dan desain

Catatan: hindari prompt yang menampilkan klaim penyembuhan, diagnosis, atau promosi produk kesehatan secara berlebihan.
