# Komik VitaNusa AI

Folder ini digunakan untuk menyimpan konsep, naskah, storyboard, aset, dan halaman komik edukatif VitaNusa AI.

Tema utama komik:
- edukasi kesehatan yang amanah
- literasi klaim produk kesehatan
- Cek KLIK sebelum membeli
- konten keluarga Indonesia yang ramah dan tidak menakut-nakuti

Catatan: konten komik bersifat edukatif dan tidak menggantikan dokter, apoteker, ahli gizi, atau tenaga kesehatan profesional.
