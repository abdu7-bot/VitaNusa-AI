# Naskah Komik

Folder ini digunakan untuk menyimpan naskah komik edukatif VitaNusa AI.

Isi yang cocok disimpan di sini:

- dialog karakter
- narasi pendek
- pesan edukasi
- catatan keamanan kesehatan
- versi revisi naskah

Prinsip konten:

- edukatif, bukan diagnosis
- tidak menjanjikan kesembuhan
- tidak membuat klaim produk berlebihan
- menyarankan konsultasi tenaga kesehatan untuk keluhan serius atau menetap
