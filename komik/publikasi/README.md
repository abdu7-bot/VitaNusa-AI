# Publikasi Komik

Folder ini digunakan untuk menyimpan hasil akhir komik VitaNusa AI yang siap dipakai.

Isi yang cocok disimpan di sini:

- versi final komik
- file untuk website
- file untuk media sosial
- caption publikasi
- arsip versi komik yang sudah selesai

Sebelum publikasi, pastikan komik tetap bersifat edukatif, tidak mendiagnosis, tidak menjanjikan kesembuhan, dan menyarankan konsultasi tenaga kesehatan untuk keluhan serius.
