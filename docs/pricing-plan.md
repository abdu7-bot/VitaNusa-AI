# Rencana Paket Berbayar VitaNusa AI

Dokumen ini menjelaskan rencana paket berbayar VitaNusa AI. Harga dan fitur masih dapat disesuaikan berdasarkan biaya API, kebutuhan pengguna, dan validasi pasar.

## Prinsip Harga

Paket berbayar VitaNusa AI harus jelas dan amanah:

1. Pengguna tahu apa yang mereka dapatkan.
2. Batas pemakaian dijelaskan dengan jujur.
3. Tidak menjual akses API key.
4. Tidak menjual akun ChatGPT bersama.
5. Tidak menyebut VitaNusa AI sebagai produk resmi OpenAI jika belum ada kerja sama resmi.
6. AI diposisikan sebagai fitur edukasi, bukan dokter atau alat diagnosis.

## Struktur Paket

| Paket | Target | Kisaran Harga | Fungsi Utama |
|---|---|---:|---|
| Gratis | Pengunjung umum | Rp0 | Artikel, VitaCheck dasar, edukasi ringan |
| Plus | Pengguna aktif | Rp59.000 - Rp79.000/bulan | Chat edukasi AI, rangkuman artikel, VitaCheck lanjutan |
| Pro | Pengguna serius | Rp179.000 - Rp249.000/bulan | Analisis label produk, dokumen panjang, rencana kebiasaan |
| Ultra | Reseller, edukator, UMKM, tim | Rp399.000 - Rp599.000/bulan | Analisis klaim, laporan PDF, konten premium, dashboard tim |

## Paket Gratis

Untuk pengguna awal yang ingin mengenal VitaNusa AI.

Fitur:

- Baca artikel edukasi
- VitaCheck dasar
- Produk pilihan
- FAQ
- Kontak WhatsApp

Batasan:

- Tidak ada chat AI premium
- Tidak ada analisis gambar
- Tidak ada export PDF

## Sistem Kredit

Sistem kredit digunakan agar biaya API tetap terkendali.

Contoh top-up:

| Top-up | Harga |
|---|---:|
| 30 kredit premium | Rp29.000 |
| 75 kredit premium | Rp59.000 |
| 150 kredit premium | Rp99.000 |

## Model Routing

Jangan semua fitur memakai model paling mahal.

Rekomendasi:

- Artikel pendek dan FAQ: model ringan atau cache.
- Rangkuman pendek: model standar.
- Analisis label produk, dokumen panjang, dan klaim produk: model premium.
- Fitur yang sering dipakai banyak orang: wajib dibatasi dengan kredit.

## Catatan Akhir

Harga jangan terlalu murah jika fitur memakai model premium. Murah di depan tapi rugi di belakang bukan strategi, itu latihan sabar yang tidak perlu.
